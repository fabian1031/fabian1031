## 👨‍💻 Sobre mí

Estudiante de **Ingeniería de Software** en el Politécnico Grancolombiano, con formación complementaria en el **Bootcamp Full-Stack Java de Generation Colombia**. Me especializo en el lado backend — APIs robustas, bases de datos relacionales y despliegues en producción. También manejo el frontend lo suficiente para entregar productos completos cuando el proyecto lo requiere.

Antes de enfocarme en el desarrollo, trabajé en **Soporte IT corporativo**, desplegando aplicaciones en Windows Server e IIS — una base práctica que me da una visión diferente a la de un desarrollador sin experiencia en infraestructura.
